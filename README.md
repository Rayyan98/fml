# fourminutes.life

A minimal blog. Short reads about tech, life, and ideas.

## Development

```bash
npm run dev
```

## Adding a post

Create a new `.mdx` file in `content/posts/`:

```yaml
---
title: "Your Title"
date: "2026-02-19"
description: "A short description."
tags: ["tag1", "tag2"]
---

Your content here in MDX.
```

## Build

```bash
npm run build
```
